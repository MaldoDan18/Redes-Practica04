# Redes-Practica04

Repositorio para la práctica 4 de Redes de Computadoras.

Programar una aplicación servidor HTTP que atienda peticiones de tipo GET, POST, PUT, DELETE para al menos 4 tipos MIME distintos. 

El servidor contará con un pool de conexiones(tamaño definido por el usuario) y cuando el pool supere la mitad del tamaño se deberá iniciar un segundo servidor y redireccionar las peticiones al segundo servidor.
